# Terminal chat

A realtime chat made using NodeJS for using inside the terminal.

## Demo

![Demo of the application](public/demo.gif)

## How to run?

1. Install the dependencies

```sh
$ npm i
```

2. Start the server

```sh
$ npm run start:server
```

3. Start a client

```sh
$ npm run start:client
```
